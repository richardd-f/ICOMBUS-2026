ICOMBUS-2026 is the official static website for the International Conference on Management, Business, and Social Science. Built with Next.js, this platform provides essential conference information, including schedules, speaker lineups, and registration details.

The project is designed for high performance and responsiveness, leveraging a modern frontend stack to ensure a seamless experience for conference attendees and presenters.
